# Kali Linux Autonomous Swarm (Pi 5 Edition)

An autonomous, self-improving AI agent swarm designed specifically for a Raspberry Pi 5 running Kali Linux. The Swarm uses a local, uncensored LLM (`dolphin-phi`) via Ollama to intelligently orchestrate physical security, wireless auditing, OSINT, and exploitation tools. It features a secure Discord Command & Control (C2) interface for remote operation.

## 🚀 Features
- **Local & Uncensored**: Runs entirely on the Pi 5's ARM CPU using optimized 4-bit quantization, requiring no external API keys (except Discord).
- **Discord C2**: Send commands securely from your phone or desktop via a strictly authenticated Discord Bot.
- **Dynamic Skill Acquisition**: If the Swarm lacks a tool, it will dynamically write its own Python wrapper using the Gemini CLI and pause to ask for your permission before implementing it.
- **Advanced Capabilities**:
  - 📡 Automated WiFi Auditing (`wifite`, `aircrack-ng`)
  - 📻 Bluetooth/BLE Tracking & DoS (`l2ping`, `hcitool`)
  - 👿 Evil Twin / Rogue AP Deployment (`hostapd`, `dnsmasq`)
  - 🎯 Autonomous Zero-Day Scanning (`nuclei`)
  - 🔑 Offline WPA Handshake Cracking (`hashcat`)
  - 🕵️ Automated Spear-Phishing Generation
  - 🎙️ Text-to-Speech Capabilities (`edge-tts`)

## 🛠️ Installation (Foolproof)

The repository includes an interactive, foolproof installation script tailored for a fresh Kali Raspberry Pi. 

**Ensure you have an active internet connection on the Pi and run:**
```bash
chmod +x install.sh
sudo ./install.sh
```

The script will automatically grab dependencies, install Ollama, download the LLM, set up the Python `venv`, and securely prompt you for your keys to configure the `.env` file.

## 🎮 Usage 

### 1. Discord Command & Control (Recommended)
This runs the lightweight async bot that listens for commands from your specific Discord User ID.
```bash
source venv/bin/activate
sudo python3 discord_bot.py
```
*Note: `sudo` is highly recommended so the agent has permission to alter network interfaces and run packet injections.*

### 2. Local CLI Mode
If you do not want to use Discord, you can assign single objectives directly to the Swarm manager via the terminal:
```bash
source venv/bin/activate
sudo python3 main.py --task "Run an aggressive nmap scan on 192.168.1.0/24 and report back any open FTP ports."
```
